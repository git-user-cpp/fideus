2) üóÇÔ∏è Download *all* files and put them in a one directory.

3) ‚öíÔ∏è Compile and run the program with *cargo*.
    - –°ompilation example
    ```
    cargo run
    ```
4) üë∑ Choose the first function by inserting **1**, then enter the number of products, product names and prices.
5) :ballot_box_with_check: To view the list of products, select the **2** function in the main menu *(after entering the
   products)*.
6) üå† To view the total sum and percentage, select the **3** function in the main menu *(after entering the products)*.
